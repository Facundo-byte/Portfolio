## Resumen

Desarrollo de una aplicación web interactiva de carrito de compras utilizando React para la lógica de componentes, Tailwind CSS para el diseño y Framer Motion para animaciones fluidas e intuitivas. El proyecto destaca por su arquitectura simple, modular y enfocada en la experiencia del usuario.

## Caracteristicas principales

- **Componentes reutilizables en React**: Construcción de una interfaz completamente modular basada en componentes funcionales, lo que permite una estructura escalable y fácil de mantener.
- **Carrito de compras totalmente funcional**: Implementación de funciones para agregar productos, modificar cantidades, eliminar ítems y calcular totales automáticamente.
- **UI moderna con Tailwind CSS**: Diseño responsivo y limpio usando utilidades de Tailwind, permitiendo iterar rápidamente en el estilo sin escribir CSS tradicional.
- **Animaciones con Framer Motion**: Uso de animaciones suaves en la aparición, interacción y eliminación de productos, logrando una experiencia visual atractiva sin afectar el rendimiento.

## Tecnologias usadas

- **React**: Base del proyecto y manejo de estado mediante hooks.
- **Tailwind CSS**: Librería utility-first para estilos rápidos, responsivos y consistentes.
- **Framer Motion**: Para animaciones fluidas y declarativas.
- **Figma**: Para el maquetado de la web.

## Resultado

El resultado fue una aplicación ligera, rápida y estéticamente atractiva. El carrito funciona de forma fluida y confiable, con transiciones suaves y una experiencia de usuario consistente. El proyecto sirve como una base sólida para expandirse hacia un e-commerce completo, agregando funcionalidades como filtros, login, entre otros.
